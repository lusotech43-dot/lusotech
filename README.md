# LusoTech

Engineered Digital Solutions — corporate website for LusoTech, a Portuguese software development studio.

## Tech Stack

- Vanilla HTML5, CSS3, JavaScript (no frameworks, no build tools)
- Client-side i18n engine (EN/PT translations)
- Dark/light theme with system preference detection
- SVG icon system (lazy-loaded partials)
- Hosted on GitHub Pages

## Pages

| Page | Description |
|---|---|
| Home | Hero, services, testimonials, contact form |
| About | Company story |
| Portfolio | Project showcase with category filters and lightbox |
| Products | Product catalog with purchase modal |

## Development

This is a static site. Open any `.html` file in your browser to view.
